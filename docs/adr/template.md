## Instructions for Use

1. **Copy this template** to create a new ADR
2. **Number sequentially**: Use the next available number (e.g., 0003, 0004, etc.)
3. **Replace placeholders**: Fill in all sections marked with [brackets]
4. **Keep it concise**: ADRs should be readable in 5-10 minutes
5. **Focus on "why"**: Explain the reasoning, not just what was decided
6. **Update status**: Change status as the decision evolves
7. **Link related ADRs**: Reference other ADRs when relevant

### Status Guidelines

- **Proposed**: Decision is under discussion
- **Accepted**: Decision has been approved and is active
- **Deprecated**: Decision is no longer recommended but may still be in use
- **Superseded**: Decision has been replaced by a newer ADR (reference the new one)
